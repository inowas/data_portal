Add new features

Instructions for file upload: A *.geojson file containing multiple features can be uploaded. Supported features: polygon, point, linestring. Feature's attributes should contain following fields: "name" - text value that will be assigned to a new feature and "type" - text value representing type of the created feature. "Type" fields should contain one of the following: "sampling_feature:station", "sampling_feature:profile", "sampling_feature:area", "hydrography:stream", "hydrography:junction", "hydrography:lake", "hydrography:pond", "hydrography:reservior", "hydrography:watershed", "hydrography:water_divide", "hydrogeology:aquifer", "hydrogeology:aquitard", "hydrogeology:water_divide", "hydrogeology:layer", "hydrogeology:well", "other".
